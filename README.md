# 🧮 Calculadora Web Simple

Esta es una calculadora básica construida con **HTML**, **CSS** y **JavaScript**, que permite realizar operaciones aritméticas simples a través de una interfaz limpia y funcional.


## 🧰 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ⚙️ Funcionalidades

- Interfaz de usuario sencilla y responsive.
- Botones funcionales para:
  - Números del 0 al 9
  - Doble cero (00)
  - Operadores aritméticos: `+`, `-`, `*`, `/`
  - Punto decimal `.`
  - Botón AC (limpia toda la operación)
  - Botón DE (borra el último carácter)
  - Botón `=` para calcular el resultado
